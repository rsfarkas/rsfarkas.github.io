    • Relaxation Canoe

    Design Question

    How might we help those who are unable to take long breaks during buy days relax at their desks?

    Concept

    City life is stressful and people need an escape without leaving their desks. In order to combat the lack of time and calming environments, Relaxation Canoe creates an interactive nature experience to allow people to have an escape into the wilderness without ever needing to go anywhere.The user is able to walk around, swim underwater, or go for a ride inside of a canoe while listening to calming nature sounds.

    Implementation

    • -Terrain created through GAIA in combination with Unity's built in terrain engine.
    • -C++ Script to control first person character
    • -Soundtrack attached to main camera
    • -C++ Script to create an underwater fog effect
    • -C++ Script to control canoe object

    Product

    Using the arrow controls on the keyboard, the user is able to walk around the terrain. The terrain is made realistic through physics elements including wind, light, and sound.

    In order to guide the user toward the canoe, a giant glowing ball moves slowly through the sky. The user is also free to roam through the forests or go for a swim under the lake. Should they choose to get into the canoe they are able to steer it using the same arrow controls and go for a ride through the lake.
